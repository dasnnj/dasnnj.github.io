git生成ssh 公钥-windows

发表于 2019-02-21 更新于 2019-02-25 分类于 tool
  • 首先可以查看是否已经有了config（当然有了也可以覆盖掉）
  • 可以通过git config user.name （email类似）
  • 如过没有设置git config信息的话先设置用户名和邮箱，通过git config user.name “用户名” 和 git config user.email “邮箱”
    （以上设置的是局部信息，设置全局可使用git config –global user.name ，eamil方式和name一样）
    设置的信息将会体现在提交代码的提交人信息

  • 可以通过git config user.name （email类似）查看设置好了没

  • 之后通过ssh-keygen -t rsa -C “邮箱”生成，后面可以一路回车或者可以在输入加密密码的时候输入一下。具体信息可自行百度

  • 生成的公钥存在C:\Users\你的用户名\.ssh里面的以 .pub 结尾的文件里面，随便以文本格式打开，复制下来分别复制到github,和dev.tencent
